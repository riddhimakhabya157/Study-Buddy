## About
Study Buddy is a web-based AI-powered student assistant designed to help students:

- Find the right study resources for any topic
- Get instant answers to common doubts (Newton, Circle, Photosynthesis)
- Stay motivated with quotes
- Focus using simulated focus music
- Manage time with a simple timer

**Target Users:** School and college students  
**Goal:** Improve productivity, time management, and mental well-being
